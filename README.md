# ANU-MACYB-public

Public repository for **Algoplexity** — an evolving research programme focused on
rigorous, computable approaches to intelligence, coordination, structure, and change
in complex systems.

This repository hosts the public-facing artefacts of the programme, including the
**Collective Intelligence Observatory (CIO)**, the first major instrument concept
brought into public view.

**Project site:** https://algoplexity.github.io/ANU-MACYB-public/

---

## Algoplexity

Algoplexity is an evolving research programme that builds computable, measurable
frameworks for understanding intelligence and coordination in complex systems.

The programme spans theory, computation, and instrumentation — with particular
emphasis on algorithmic cognitive science, cybernetics, collective intelligence,
and mesoscopic measurement.

The Cybernetic Intelligence (CI) synthesis is the theoretical anchor of the
programme. It treats collective intelligence not as a vague behavioral quality,
but as a measurable coordination phenomenon with two separable components:
**interaction structure** and **interaction dynamics**.

## Collective Intelligence Observatory (CIO)

The Collective Intelligence Observatory is the flagship instrument concept within
Algoplexity. Its goal is to make collective intelligence measurable as a real,
observable quantity.

### Core idea

Collective intelligence is not just a behavioral or cultural phenomenon — it can
be approached as a physically measurable property of a group.

CIO proposes measuring it through two complementary signals:

- **Interaction structure** — the topology of who interacts with whom, captured
  as a dynamic interaction graph
- **Interaction dynamics** — how that topology forms, stabilises, and evolves,
  captured through motion and proximity fields

Together, these signals provide a real-time estimate of collective coordination.

### Coordination regimes

The CIO framework distinguishes four coordination states:

| Regime | Description |
|---|---|
| True coordination | Structured, stable, compressible interaction topology |
| False coordination | Apparent structure that does not persist or generalise |
| Fragile coordination | Structure present but sensitive to perturbation |
| No coordination | Interaction topology shows no compressible pattern |

### Falsifiability

The theory makes a testable prediction: groups that are genuinely coordinating
should produce interaction structures that become more compressible over time.
If coordinated groups do not show increased compressibility in interaction
structure, the theory is falsified.

## Repository structure

This repository also hosts curated, sanitised public artefacts from the
**Master of Applied Cybernetics (MACYB)** at the Australian National University,
which provides the research context for the programme.

| Location | Contents |
|---|---|
| `docs/` | GitHub Pages source — site home, blog, and CIO overview |
| `projects/cps1-individual/` | CPS1 individual project artefacts |
| `projects/cps2-group/` | CPS2 group project artefacts |

GitHub Pages is served from the `/docs` directory on the `main` branch.

```
docs/               # GitHub Pages source (Jekyll/Markdown)
  blog/             # Blog posts
  pages/            # Static pages (About, CIO overview)
  assets/images/    # Images used by the site
projects/
  cps1-individual/  # CPS1 individual project artefacts
  cps2-group/       # CPS2 group project artefacts
```

## Intentionally excluded

The following will **never** appear in this public repository:

- Assessment submissions (reports, essays, exams)
- Raw personal reflections or journal entries
- Sensitive or private data of any kind
- Large or raw datasets
- Any information that could identify peers or staff
