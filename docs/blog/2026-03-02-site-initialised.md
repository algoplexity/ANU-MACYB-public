# Site initialised

The public repository and GitHub Pages site for the
**Master of Applied Cybernetics (MACYB)** at the Australian National University
are now live.

This site will host sanitised code samples, project overviews, and occasional
posts on topics explored during the programme. It is a curated public subset —
assessment artefacts, raw data, and private material remain off-limits.

More content will appear as projects progress and artefacts are reviewed for
publication.
